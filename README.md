# Solar Power Bank with Wireless Charging

## Description
A portable power bank that charges via solar energy and supports Qi wireless charging for compatible devices. Eco-friendly solution for outdoor use & power outages.

## Components Used
- Arduino Uno/Nano
- Solar Panel 6V
- TP4056 Charging Module  
- 18650 Li-ion Battery
- Qi Wireless Charging Module
- 5V Boost Converter

## Features
- Solar charging capability
- Wireless charging output
- USB charging output
- Battery level indicator

## How it Works
Solar panel → TP4056 → Battery → Boost Converter → Wireless/USB Output
